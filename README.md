# Raicici Radovan - Portfolio Personal

Site de prezentare personală realizat cu **React** și **Tailwind CSS**, conectat la **Strapi CMS** și găzduit pe **Netlify**.

## 🌐 Link-uri

- **Site live:** https://echipa-15-3ie26-proiect2.netlify.app/
- **Strapi CMS:** https://raicici-radovan.strapiapp.com
- **Repository GitHub:** https://github.com/echipa-15-3ie26-proiect2

## 📄 Pagini

| Pagina | Descriere | Layout |
|--------|-----------|--------|
| **Home** | Hero banner, skill-uri, proiecte recente, preview blog, CTA | Two-column hero + multi-section |
| **About** | Biografie, timeline, interese, fun facts | Single Type - portrait layout |
| **Projects** | Grilă proiecte cu filtre pe categorie | Grid cu filtre |
| **Blog** | Lista articole cu tag-uri și newsletter | List + featured |
| **Contact** | Formular de contact + date contact | Split layout |

## 🛠 Tehnologii

- **Frontend:** React 18, React Router v6, Tailwind CSS v3
- **Fonturi (tweakcn.com):** Playfair Display, DM Sans, JetBrains Mono
- **CMS:** Strapi (Headless)
- **Hosting:** Netlify
- **Build tool:** Vite

## 📦 Instalare locală

```bash
# Clonare repository
git clone https://github.com/echipa-15-3ie26-proiect2.git
cd echipa-15-3ie26-proiect2

# Instalare dependențe
npm install

# Rulare în mod development
npm run dev

# Build pentru producție
npm run build
```

## 🎨 Tema tweakcn.com

Link temă: https://tweakcn.com/editor/theme?theme=...

Fonturi utilizate:
- **Playfair Display** – titluri și headings (font display)
- **DM Sans** – text corp și navigare (font body)
- **JetBrains Mono** – cod, badge-uri, etichete (font mono)
